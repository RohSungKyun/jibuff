MCP stdio server (jibuff_mcp/server.py) registered in Claude Code via `jb setup`. Exposes four tools: jibuff_interview (start/continue an interview session, persists state as workspace-local markdown files with optimistic concurrency via revision numbers), jibuff_run (execute LoopController), jibuff_status (read ArtifactStore), jibuff_cancel (write cancellation artifact). Parent-process watcher exits the server if Claude Code dies.
