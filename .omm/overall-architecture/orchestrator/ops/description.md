orchestrator/ops.py provides helper functions for CLI utility commands: run_doctor (checks prerequisites like API keys, claude/codex on PATH, spec file), inspect_workspace (aggregates task status + interview session + runtime state for `jb inspect`), cleanup_workspace (removes expired MCP session files and orphan locks), recover_workspace (requeues stale in-progress tasks), install_skill (writes SKILL.md for Codex discovery).
